# 여행영어 작업 기록

## 2026-07-20 — 프로젝트 시작, v1 완성
- 기술 방향 확정: 바닐라 HTML/CSS/JS + GitHub Pages(무료), 런타임 API 없음
- 여행영어 50문장 초안 작성 (5주 과정: 기본패턴 → 공항 → 호텔·교통 → 식당·쇼핑 → 문제해결·긴급)
- Edge TTS(무료)로 음원 100개 생성 (느리게/보통 × 50문장, 총 1.8MB)
- 앱 v1 완성:
  - 프로필 선택 (엄마/아빠/동생) + 기기별 진도·출석 저장 (localStorage)
  - 학습 플로우: 듣고 이해 → 따라 말하기(4초 카운트다운) → 한국어만 보고 말하기 → 자가평가
  - 간격 반복(다음날/3/7/14/30/60일), "다시 연습할래요"는 세션 내 재출제 + 내일 재복습
  - 하루 새 문장 3개 + 복습 최대 8개, 출석 도장·연속일·주간 출석 점
  - 문장집(전체/배운 문장/즐겨찾기), 바꿔 말하기 팁, 상대방 대사(cue), 내 목소리 녹음
- 브라우저에서 전체 플로우 테스트 완료
- 남은 일: Amy 문장 검토, 캐릭터 이미지 제작·교체, GitHub 배포, (2단계) 카톡 알림

## 2026-07-20 — 말틀(패턴) 기반 구조로 재설계
- Amy 피드백 반영: "50개 문장 암기"에서 "10개 핵심 말틀 × 4개 여행 활용 문장(40) + 통째 암기 필수표현 10" 구조로 전환
- data.js에 PATTERNS(말틀 10개: I'd like/Can I/Could you/Do you have/Where is/How much is/I need/I have/I'm looking for/Is there) 추가, 문장마다 pattern 연결 + patternIntro(처음 등장) 표시
- 같은 말틀이 연속으로 나오지 않도록 주차 전체를 상황 섞어서 재배치 (예: I'd like → 카페 → 식당 → 쇼핑 → 기내로 며칠에 걸쳐 재등장)
- 학습 화면: 말틀 고정부(주황)·교체부(파랑) 색상 구분, 말틀 첫 등장 시 "이렇게도 말해요" 미리보기 칩(4개 활용 탭하면 음성 재생), 재등장 시 "전에 배운 말틀이에요" 리마인드 태그
- 복습 단계 힌트 점진적 축소: 처음 배울 때는 한국어+말틀 힌트("I'd like ___") 함께 표시 → 복습 초반(레벨 0~1)은 한국어만 → 숙련 복습(레벨 3+)은 상황만 보여주고 한국어는 버튼 눌러야 보이게
- 음원 100개 전량 재생성 (문장 내용이 바뀌어서 기존 mp3 삭제 후 재생성)
- 브라우저에서 패턴 인트로/재등장/힌트단계별 렌더링 모두 확인 완료
- 검토: PWA(홈 화면 추가) 웹 푸시로 아침 알림도 기술적으로 가능하지만 결국 무료 서버 트리거가 필요해 카톡 알림(2단계 계획)과 복잡도가 같음 → 카톡 경로 하나로 유지하기로 함

## 2026-07-20 — 개인용(어머님) / 공유용 두 버전으로 분리
- 이 저장소(여행영어)는 **시어머님 문숙님 전용**으로 전환:
  - 시작 화면을 개인화 — ✈️☁️🧳 장식 + 어머님 캐릭터 사진 + "문숙님의 여행회화 ✈️" + "비행기 타기 전에 하루 10분씩 같이 해봐요! 😊"
  - 하단에 작게 "💝 최윤주가 만든 어머님 공부 선물" 표시 (data.js의 OWNER에서 이름·문구 수정 가능)
  - 프로필을 1명(문숙)으로 줄이고, 시작 화면이 곧 환영 화면이 되도록 함 (진도가 있으면 "지금까지 N문장 배우셨어요 👏" 표시)
  - 대시보드 상단 버튼 "👥 사용자 변경" → "‹ 처음으로"
  - 어머님 캐릭터 사진은 `img/munsook.png`로 넣으면 자동 적용, 없으면 이모지(👵🏻)로 대체 (원형 크롭 + 응원 배지 유지)
- 공유용 별도 폴더/저장소 생성: `여행영어-배포용` → https://github.com/NabiLab-dev/travel-english-app
  - 첫 화면에서 "✏️ 이름 바꾸기 · 사람 추가"로 사용자 편집 가능 (이름 수정, 이모지 순환 선택, 삭제, 추가 — 최대 3명)
  - 이름을 바꿔도 profile id가 유지되어 기존 학습 기록이 보존됨(테스트 확인). 삭제 시에만 해당 기록 삭제
  - 개인정보(문숙/최윤주/Amy/CLAUDE.md/CHANGELOG.md 등)는 공유용에 포함하지 않음 — grep으로 검사 완료
  - 공유용 README.md 작성 (앱 소개, 사용법, 포크해서 고치는 법)

## 2026-07-21 — 어머님 캐릭터 이미지 적용 + 폴더 정리

  - 응원/축하 이모지 배지 위치도 전신 이미지 기준으로 재조정
- 브라우저에서 시작화면·대시보드·완료화면 세 곳 모두 이미지 로딩과 크기 확인 완료
