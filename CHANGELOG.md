## 2026-07-20 — 말틀(패턴) 기반 구조로 재설계
- Amy 피드백 반영: "50개 문장 암기"에서 "10개 핵심 말틀 × 4개 여행 활용 문장(40) + 통째 암기 필수표현 10" 구조로 전환
- data.js에 PATTERNS(말틀 10개: I'd like/Can I/Could you/Do you have/Where is/How much is/I need/I have/I'm looking for/Is there) 추가, 문장마다 pattern 연결 + patternIntro(처음 등장) 표시
- 같은 말틀이 연속으로 나오지 않도록 주차 전체를 상황 섞어서 재배치 (예: I'd like → 카페 → 식당 → 쇼핑 → 기내로 며칠에 걸쳐 재등장)
- 학습 화면: 말틀 고정부(주황)·교체부(파랑) 색상 구분, 말틀 첫 등장 시 "이렇게도 말해요" 미리보기 칩(4개 활용 탭하면 음성 재생), 재등장 시 "전에 배운 말틀이에요" 리마인드 태그
- 복습 단계 힌트 점진적 축소: 처음 배울 때는 한국어+말틀 힌트("I'd like ___") 함께 표시 → 복습 초반(레벨 0~1)은 한국어만 → 숙련 복습(레벨 3+)은 상황만 보여주고 한국어는 버튼 눌러야 보이게
- 음원 100개 전량 재생성 (문장 내용이 바뀌어서 기존 mp3 삭제 후 재생성)
- 브라우저에서 패턴 인트로/재등장/힌트단계별 렌더링 모두 확인 완료
- 검토: PWA(홈 화면 추가) 웹 푸시로 아침 알림도 기술적으로 가능하지만 결국 무료 서버 트리거가 필요해 카톡 알림(2단계 계획)과 복잡도가 같음 → 카톡 경로 하나로 유지하기로 함
