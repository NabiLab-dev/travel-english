# 깃허브에 올려서 부모님께 링크 보내기

한 번만 하면 됩니다. 이후 문장/디자인 수정은 파일 고치고 다시 push만 하면 자동 반영돼요.

## 1. GitHub 저장소 만들기
1. github.com 로그인 → 우측 상단 `+` → **New repository**
2. Repository name: `travel-english` (원하는 이름)
3. **Public** 선택 → Create repository

## 2. 이 폴더를 올리기
이 폴더(`여행영어`)에서 터미널을 열고:

```bash
git init
git add index.html style.css app.js data.js audio img 배포방법.md
git commit -m "여행영어 앱 첫 배포"
git branch -M main
git remote add origin https://github.com/내아이디/travel-english.git
git push -u origin main
```

(Claude Code에게 "깃허브에 올려줘"라고 하면 대신 해줍니다)

## 3. GitHub Pages 켜기
1. 저장소 페이지 → **Settings** → 왼쪽 메뉴 **Pages**
2. Source: **Deploy from a branch**, Branch: **main** / root → Save
3. 1~2분 뒤 `https://내아이디.github.io/travel-english/` 주소가 생김

## 4. 부모님 폰에 설치처럼 만들어드리기
1. 카톡으로 위 주소 전송
2. 부모님 폰에서 링크 열기
3. **아이폰**: 사파리 공유 버튼 → "홈 화면에 추가"
   **갤럭시**: 크롬 메뉴(⋮) → "홈 화면에 추가"
4. 홈 화면에 앱 아이콘이 생겨서 앱처럼 쓸 수 있어요

## 자주 하는 수정
- **문장 수정**: `data.js`에서 고치기 → 해당 음원 mp3 삭제 → `python tools/generate_audio.py` 재실행 → git push
- **사용자 이름 변경**: `data.js` 맨 위 PROFILES에서 이름/이모지 수정
- **하루 학습량 조절**: `app.js` 맨 위 `NEW_PER_DAY`(새 문장 수), `MAX_REVIEWS`(복습 수) 숫자 변경
