# 여행영어 50문장 목록 (검토용)

10개 핵심 말틀 x 4개 활용문장(40개) + 통째로 외우는 필수표현 10개 = 50문장.
수정할 문장이 있으면 번호와 바꿀 내용만 알려주세요.

## 핵심 말틀 10개

| 말틀 | 뜻 |
|---|---|
| I'd like | ~ 주세요 |
| Can I | 제가 ~해도 될까요 |
| Could you | ~해 주시겠어요 |
| Do you have | ~ 있나요 |
| Where is | ~ 어디예요 |
| How much is | ~ 얼마예요 |
| I need | ~ 필요해요 |
| I have | ~ 있어요 |
| I'm looking for | ~ 찾고 있어요 |
| Is there | ~ 있나요 |
